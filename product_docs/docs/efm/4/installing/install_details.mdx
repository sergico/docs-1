Components are installed in the following locations, where `4.x` indicates a minor release.

| Component                         | Location                      |
| --------------------------------- | ----------------------------- |
| Executables                       | `/usr/edb/efm-4.x/bin`        |
| Libraries                         | `/usr/edb/efm-4.x/lib`        |
| Cluster configuration files       | `/etc/edb/efm-4.x`            |
| Logs                              | `/var/log/efm-4.x`            |
| Lock files                        | `/var/lock/efm-4.x`           |
| Log rotation file                 | `/etc/logrotate.d/efm-4.x`    |
| sudo configuration file           | `/etc/sudoers.d/efm-4.x`      |
| Binary to access VIP without sudo | `/usr/edb/efm-4.x/bin/secure` |
